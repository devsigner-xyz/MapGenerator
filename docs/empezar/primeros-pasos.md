# Recorrido rapido

Esta pagina resume el flujo mas corto para pasar de cero a explorar la app.

## 1. Entiende que vas a abrir

Nostr City no es un cliente de timeline clasico. Es una capa visual para leer relaciones, actividad y proximidad social dentro de Nostr.

## 2. Decide como quieres entrar

Puedes usar una identidad existente o crear una cuenta local, segun el tipo de acceso soportado por la version actual de la app.

## 3. Ajusta relays si lo necesitas

La experiencia depende de los relays desde los que cargas perfiles, follows y actividad. Si vienes con relays propios, te conviene revisarlos.

## 4. Explora la ciudad

Una vez dentro, la app construye una representacion espacial del ecosistema y te deja navegar perfiles, conexiones y capas visuales.
